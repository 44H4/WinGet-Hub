WinGet Hub!

The best place to download programs quickly and easily (for Windows only).

These downloads are batch (.bat) scripts that contain an installer that uses WinGet to install programs using the Command Prompt/PowerShell, like in Linux

Why should I use WinGet Hub?↓
  • Easy Installation: Quickly and easily install applications with pre-made Winget batch scripts.
  • Time-Saving: Save time by automating the installation process with reliable scripts.
  • Reliable Scripts: Access well-tested and error-free Winget scripts for a seamless installation experience.
  • Community-Supported: Benefit from a community of users sharing and improving scripts for popular software.
  • Security: Download scripts from a trusted hub, ensuring safe and verified installations. (The .bat files are safe, they are open-source on Github so you can see the code (if it says Windows protected your PC, click more and run anyway)).
  • Customization: Choose from a variety of scripts tailored to different needs and preferences.
  • Cross-Platform Compatibility: WinGet Hub ensures compatibility across Windows operating systems, making installations smooth and efficient.
  • Ease of Use: You can install programs with a few clicks, unlike the frustrating experience of downloading installers, finding the right version and  clicking through 400 next buttons
  • Continuous Updates: Access the latest scripts and updates, keeping your system up-to-date effortlessly.
How to use?↓
  1. Install the program of your choice.
  2. Run the script.
  3. If you meet the requirements (WinGet installed, not already having the program, giving admin priveliges to the script, etc.) it should be installed!
